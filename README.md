# Gate-Entry

*--Starting the Node Server--*
First move to the node server directory from root directory by - 'cd nodeServer' in terminal.
install node modules by - 'npm install'
  npm install axios
  bcrypt
  body-parser
  cors
  dotenv
  express
  fast-xml-parser
  pg
  xml2js
  --save-dev nodemon

*--Starting the React UI--*
Open another Terminal
Move to the react ui folder by - 'cd reactUI'*
install node modules by - 'npm install'
Run the react ui by - 'npm run dev'
Open Browser and open 'http://localhost:5173
